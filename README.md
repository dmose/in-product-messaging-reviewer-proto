# fx-messaging-review

Claude Code plugin with review guidance for Firefox's in-product messaging system — ASRouter, about:welcome, feature callouts, infobars, spotlights, menu messages, and toast notifications.

The review skill was generated by [custom-module-reviewer](https://github.com/msujaws/custom-module-reviewer).

## Install

Add the marketplace, then install the plugin:

```
claude plugin marketplace add https://github.com/dmose/in-product-messaging-reviewer-proto
claude plugin install fx-messaging-review
```

After install, enable auto-update for this marketplace (third-party marketplaces are opted out by default):

1. Start `claude`
2. Run `/plugin` to open the plugin manager
3. Select `Marketplaces`
4. Choose `fx-messaging-review` from the list
5. Select `Enable auto-update`

See the [Claude Code docs](https://code.claude.com/docs/en/discover-plugins#configure-auto-updates) for more details.

## Usage

Invoke the review command in any Claude Code session:

```
/fx-messaging-review:review
```

This loads comprehensive review conventions covering:

- Message authoring and JSON schemas
- Targeting expressions (JEXL, camelCase conventions)
- State, storage, and multi-profile handling
- Special message actions and prefs
- Localization (Fluent)
- Telemetry and data classification
- CSS, RTL, and High Contrast Mode
- Accessibility
- JavaScript and code style
- Testing standards
- A full review checklist

## What it covers

The plugin provides domain-specific review guidance for patches touching:

- `browser/components/asrouter/`
- `browser/components/aboutwelcome/`
- `toolkit/components/messaging-system/`
- Related test, schema, localization, and documentation files

It flags common pitfalls (e.g., missing `make-schemas.py` regeneration, cached DB connections, incorrect JEXL casing) and includes campaign-specific checks for active migrations.
